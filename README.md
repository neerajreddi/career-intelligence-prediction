# 🎯 Career Intelligence Prediction System

> **A machine learning-based career recommendation system that predicts suitable career domains using ensemble learning on a custom dataset of 6,700+ student responses.**

This project helps students discover suitable career paths by analyzing their academic background, technical skills, interests, and learning preferences. Multiple machine learning models were trained and compared, with a Stacking Ensemble model achieving **86% prediction accuracy** to provide personalized career recommendations.
